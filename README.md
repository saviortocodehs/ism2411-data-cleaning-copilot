# ism2411-data-cleaning-copilot

A small Python project that cleans a messy sales dataset as part of ISM2411 assignment.

## Structure
- `data/raw/sales_data_raw.csv` - raw input (provided)
- `data/processed/sales_data_clean.csv` - output (created by script)
- `src/data_cleaning.py` - main script
- `reflection.md` - reflection on Copilot usage

## Run
From the project root:
python src/data_cleaning.py
